Rejoindre le serveur

Adresse du serveur (IP) :

dimensional-core-survival.mine.fun
Version : 1.19.2 Loader : Forge Modpack : CurseForge
  1. Installe le launcher CurseForge.
  2. Télécharge le modpack « Dimensional Core Survival » (bouton ci-dessous), puis dans CurseForge fais Create Custom Profile → Import et choisis le fichier zip.
  3. Alloue au moins 6 Go de RAM au modpack dans les réglages.
  4. Lance le jeu, va dans Multijoueur et ajoute l'IP du serveur.
  5. Bloqué ? Demande de l'aide sur le Discord.
